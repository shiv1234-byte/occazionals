# Occazionals - Luxury Fashion Rental Platform
A full-stack MERN application for renting and buying premium fashion.

## Features
- **AI Chatbot**: Integrated with Google Gemini for fashion assistance.
- **Product Catalog**: Advanced filtering (Rent/Buy) and search.
- **Admin Dashboard**: Manage inventory and orders.
- **Performance**: Lazy loading and optimized API calls.

## Tech Stack
- **Frontend**: React.js, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas
- **AI**: Google Generative AI (Gemini 1.5 Flash)

## Setup Instructions
1. Clone the repo: `git clone <your-repo-link>`
2. Install dependencies for both folders: `npm install`
3. Add your `.env` file in the server folder with `GEMINI_API_KEY` and `MONGO_URI`.
4. Run the project: `npm run dev`
